Solving Radical Equations

In this lesson, you will learn how to solve equations with radicals.

Solving radical equations is a long process of simplifying, factoring, and testing with the given values of an equation. The standard steps of solving these equations are animated in the following video:

Put into words, the steps are as follows:

1. Simplify equation by isolating values of differing powers:

√x - 3 + 5 = x
  - 5     - 5
√x - 3 = x - 5
(√x - 3)² = (x - 5)²
x - 3 = (x - 5)(x - 5)

2. Factor to simplify further:

x - 3 = (x - 5)(x - 5)
x - 3 = x² -5x - 5x +25
x - 3 = x² -10x +25
-x       -x
-3 = x² -11x +25
+3         +3
0 = x² -11x +28
0 = (x - 7)(x - 4)

3. Elimanate extraneous solutions and find real solutions:

0 = (x - 7)(x - 4)
0 = x - 7   0 = x - 4
+7   +7   +4   +4
7 = x   OR   4 = x

√(7) - 3 + 5 = (7)
√4 + 5 = 7
2 + 5 = 7
7 = 7   ✓

√(4) - 3 + 5 = (4)
√1 + 5 = 4
1 + 5 = 4
6 ≠ 4   ✗

4 is an extraneous solutionA solution of the simplified form of an equation that does not satisfy the original equation and must be discarded and is false; 7 is the true solution.
